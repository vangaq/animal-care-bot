# Питомец под контролем

**Питомец под контролем** — это учебный Telegram-бот, созданный для удобного хранения информации о питомцах, заметках и напоминаниях в одном месте.

Проект помогает пользователю быстро посмотреть карточки своих животных, сохранить важные записи и при необходимости получить подсказку от AI-модели.

## Возможности

- добавление и хранение карточек питомцев;
- указание вида питомца: кошка, собака, хомяк и другие;
- редактирование и удаление данных о питомце;
- создание заметок для каждого питомца;
- одноразовые напоминания на точную дату и время;
- просмотр профиля со списком питомцев и заметок;
- AI-чат с учётом контекста пользователя;
- AI видит имя владельца, список питомцев и заметки по ним;
- поиск полезных мест через Яндекс.Карты.

## Технологии

- **Python 3**
- **aiogram 3**
- **SQLite + SQLAlchemy**
- **LM Studio (для AI-модели)** 
- **Yandex Maps API**

## Структура проекта

- `bot.py` — точка входа;
- `handlers/` — сценарии и обработчики;
- `db/` — модели и запросы к базе данных;
- `keyboards/` — клавиатуры Telegram;
- `utils/` — AI-клиент, напоминания, Яндекс.Карты и вспомогательные функции;
- `config.py` — настройки проекта;
- `.env` — переменные окружения.
